Car Parking Management System (C++)
Overview

A console-based Car Parking Management System built using C++ that simulates real-world parking operations such as car entry, record management, expense calculation, and checkout.

This project demonstrates a strong understanding of Object-Oriented Programming, file handling, and system design logic.

Features
Secure Login System
Password-based authentication with masked input
Park a Car
Add driver name, car name, number plate, and parking duration
View Car Details
Retrieve stored car data using a unique Car ID
Expense Calculation
Automatically calculates parking cost (₹100 per hour)
Remove Car (Checkout)
Deletes car records from the system
File Handling
Stores data persistently using text files
